## O que é o [[SITE_NAME]]?

[[SITE_NAME]] é um CMS headless de código aberto orientado por agentes, construído sobre o T3 Stack. Combina Next.js, tRPC, Drizzle ORM e Better Auth num sistema de gestão de conteúdos coeso, otimizado para desenvolvimento assistido por IA.

### Funcionalidades principais

- **Desenvolvimento orientado por agentes** — O CLAUDE.md serve como guia abrangente do projeto, permitindo que agentes de IA compreendam e modifiquem a base de código de forma eficaz
- **Stack moderno** — Construído com Next.js 16, TypeScript e Tailwind CSS v4
- **Conteúdo flexível** — Páginas, publicações de blog, elementos de portfólio, cartões vitrine, categorias e etiquetas
- **Acesso baseado em funções** — Funções de utilizador, editor, administrador e superadministrador com permissões baseadas em políticas
- **Gestão de média** — Carregamento, organização e distribuição de ficheiros multimédia com miniaturas automáticas
- **Otimizado para SEO** — Meta-descrições, imagens OG, JSON-LD, sitemaps dinâmicos e redirecionamentos de slug
- **Primitivas SaaS** — Organizações, faturação Stripe, notificações, tempo real com WebSocket
