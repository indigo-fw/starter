%SITE_NAME%（以下「本サービス」）は、%COMPANY_NAME%（所在地: %COMPANY_ADDRESS%）が運営しており、Cookieおよび類似の追跡技術を使用しています。本Cookieポリシーでは、Cookieとは何か、当社がCookieをどのように使用するか、Cookieに関するお客様の選択肢、およびCookieに関するその他の情報について説明します。

**1. Cookieとは？**

Cookieとは、ウェブサイトを訪問した際にお客様のデバイス（コンピューター、タブレット、またはモバイル端末）に保存される小さなテキストファイルです。Cookieは、ウェブサイトの効率的な動作、アナリティクス情報の提供、および特定の機能の有効化のために広く使用されています。当社は、以下に説明する特定の機能目的のためにlocalStorageも使用しています。

**2. 当社が使用するCookieの種類**

**2.1 厳密に必要なCookie**

これらのCookieは本サービスの運営に不可欠です。これらのCookieがなければ、本サービスは正常に機能しません。これらのCookieにはお客様の同意は必要ありません。

| Cookie                               | 目的                               | 期間   |
| ------------------------------------ | ---------------------------------- | ------ |
| `__Secure-better-auth.session_token` | 認証済みセッションの維持           | 1年    |
| Cookie同意Cookie                      | Cookie同意設定の保存               | 1年    |

**2.2 アナリティクスCookie**

これらのCookieにより、訪問者が本サービスとどのようにやり取りしているかを理解し、情報を収集・報告することができます。これらのインサイトを本サービスの改善に役立てています。これらのCookieはGoogle Tag Manager（GTM）を通じて読み込まれます。

| Cookie   | 目的                                                           | 期間     |
| -------- | -------------------------------------------------------------- | -------- |
| `_ga`    | Google Analytics — ユニークユーザーの識別                      | 2年      |
| `_gid`   | Google Analytics — セッション内のユニークユーザーの識別        | 24時間   |
| `_gat`   | Google Analytics — リクエストレートの制限                      | 1分      |
| `_gac_*` | Google Analytics — キャンペーン情報の保存                      | 90日     |

**2.3 機能Cookie**

これらのCookieにより、拡張機能やパーソナライゼーションが可能になります。

| Cookie             | 目的                                                           | 期間   |
| ------------------ | -------------------------------------------------------------- | ------ |
| テーマ設定         | お好みのカラーテーマ（ライト/ダーク）の保存                    | 1年    |
| 言語設定           | お好みの言語/ロケールの保存                                    | 1年    |

**2.4 ローカルストレージ**

当社は、以下の目的でブラウザのlocalStorage（HTTPCookieではありません）も使用しています。

| キー               | 目的                                                           |
| ------------------ | -------------------------------------------------------------- |
| テーマ設定         | セッション間でライト/ダークモードの設定を保持                  |
| ユーザー設定       | サイドバーの状態などのUI設定を保存                             |

**3. サードパーティCookie**

一部のCookieは、当社のページに表示されるサードパーティサービスによって設定されます。当社はこれらのCookieを管理していません。Cookieを設定する可能性のあるサードパーティには以下が含まれます。

- **Google Analytics / Google Tag Manager** — アナリティクスおよびコンバージョントラッキング
- **Cookie同意プロバイダー** — Cookie同意管理
- **決済処理業者** — 決済処理中に必要な機能Cookie

各サードパーティサービスには独自のプライバシーポリシーおよびCookieポリシーがあります。確認されることをお勧めします。

**4. Cookie同意管理**

本サービスに初めてアクセスした際、Cookie同意バナーが表示されます。必須でないCookieカテゴリーを承認または拒否することができます。

Cookieの設定は、本サービス上に引き続きアクセス可能なCookie設定ウィジェットをクリックすることで、いつでも変更できます。

ブラウザの設定からCookieを制御することもできます。

- **Chrome:** 設定 > プライバシーとセキュリティ > Cookie
- **Firefox:** 設定 > プライバシーとセキュリティ > Cookie
- **Safari:** 環境設定 > プライバシー > Cookie
- **Edge:** 設定 > プライバシー、検索、サービス > Cookie

厳密に必要なCookieを無効にすると、本サービスが正常に機能しなくなる可能性がありますのでご注意ください。

**5. 同意の撤回方法**

必須でないCookieに対する同意は、以下の方法でいつでも撤回できます。

- 本サービス上のCookie設定ウィジェットをクリックする
- ブラウザのCookieを消去し、本サービスを再訪問する（新しい同意バナーが表示されます）
- ブラウザ設定でCookieの設定を調整する

**6. 本Cookieポリシーの変更**

当社は、本Cookieポリシーを随時更新することがあります。重要な変更がある場合は、本ページに新しいCookieポリシーを掲載し、冒頭の「改定日」を更新することでお知らせします。
