## Vad är [[SITE_NAME]]?

[[SITE_NAME]] är ett agentdrivet headless CMS med öppen källkod, byggt på T3 Stack. Det kombinerar Next.js, tRPC, Drizzle ORM och Better Auth till ett sammanhållet innehållshanteringssystem som är optimerat för AI-assisterad utveckling.

### Nyckelfunktioner

- **Agentdriven utveckling** — CLAUDE.md fungerar som en omfattande projektguide som gör det möjligt för AI-agenter att förstå och modifiera kodbasen effektivt
- **Modern teknikstack** — Byggt med Next.js 16, TypeScript och Tailwind CSS v4
- **Flexibelt innehåll** — Sidor, blogginlägg, portfolioobjekt, utvalda kort, kategorier och taggar
- **Rollbaserad åtkomstkontroll** — Roller för användare, redaktör, administratör och superadmin med policybaserade behörigheter
- **Mediahantering** — Ladda upp, organisera och servera mediefiler med automatiska miniatyrbilder
- **SEO-optimerat** — Metabeskrivningar, OG-bilder, JSON-LD, dynamiska webbplatskartor och slug-omdirigeringar
- **SaaS-primitiver** — Organisationer, Stripe-fakturering, aviseringar, WebSocket-realtid
