## %SITE_NAME%이란 무엇입니까?

%SITE_NAME%은 T3 Stack 기반의 오픈 소스 에이전트 중심 헤드리스 CMS입니다. Next.js, tRPC, Drizzle ORM, Better Auth를 결합하여 AI 기반 개발에 최적화된 통합 콘텐츠 관리 시스템을 제공합니다.

### 주요 기능

- **에이전트 중심 개발** — CLAUDE.md가 종합적인 프로젝트 가이드 역할을 하여 AI 에이전트가 코드베이스를 효과적으로 이해하고 수정할 수 있도록 합니다
- **최신 기술 스택** — Next.js 16, TypeScript, Tailwind CSS v4로 구축되었습니다
- **유연한 콘텐츠** — 페이지, 블로그 글, 포트폴리오 항목, 쇼케이스 카드, 카테고리, 태그를 지원합니다
- **역할 기반 접근 제어** — 사용자, 편집자, 관리자, 최고 관리자 역할과 정책 기반 권한을 제공합니다
- **미디어 관리** — 자동 썸네일 생성과 함께 미디어 파일을 업로드, 정리, 제공합니다
- **SEO 최적화** — 메타 설명, OG 이미지, JSON-LD, 동적 사이트맵, 슬러그 리디렉션을 지원합니다
- **SaaS 기본 요소** — 조직, Stripe 결제, 알림, WebSocket 실시간 기능을 제공합니다
