## Ihr CMS ist bereit!

Dies ist eine Beispielseite, die vom Initialisierungsskript erstellt wurde. Sie können sie im [Admin-Bereich](/dashboard/cms/pages) bearbeiten oder löschen.

### Erste Schritte

- Erstellen Sie Seiten und Blogbeiträge über das Dashboard
- Laden Sie Mediendateien in die Mediathek hoch
- Konfigurieren Sie die Website-Einstellungen unter Einstellungen
- Verwalten Sie Benutzer und Rollen im Bereich Benutzer
- Richten Sie Kategorien und Schlagwörter ein, um Ihre Inhalte zu organisieren

Besuchen Sie den [Blog](/blog) für die neuesten Beiträge oder erkunden Sie das [Portfolio](/portfolio) für Projektpräsentationen.
