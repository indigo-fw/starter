## Hva er %SITE_NAME%?

%SITE_NAME% er et agentdrevet hodeløst CMS med åpen kildekode, bygget på T3 Stack. Det kombinerer Next.js, tRPC, Drizzle ORM og Better Auth til et sammenhengende innholdshåndteringssystem som er optimalisert for AI-assistert utvikling.

### Nøkkelfunksjoner

- **Agentdrevet utvikling** — CLAUDE.md fungerer som en omfattende prosjektguide som gjør det mulig for AI-agenter å forstå og modifisere kodebasen effektivt
- **Moderne teknologistack** — Bygget med Next.js 16, TypeScript og Tailwind CSS v4
- **Fleksibelt innhold** — Sider, blogginnlegg, porteføljeelementer, utvalgte kort, kategorier og tagger
- **Rollebasert tilgangskontroll** — Bruker-, redaktør-, administrator- og superadmin-roller med policybaserte tillatelser
- **Mediehåndtering** — Last opp, organiser og server mediefiler med automatiske miniatyrbilder
- **SEO-optimalisert** — Metabeskrivelser, OG-bilder, JSON-LD, dynamiske nettstedskart og slug-omdirigeringer
- **SaaS-primitiver** — Organisasjoner, Stripe-fakturering, varsler, WebSocket-sanntid
