**Revisjonsdato:** [[CURRENT_DATE]]

[[SITE_NAME]] ("Tjenesten"), drevet av [[COMPANY_NAME]], [[COMPANY_ADDRESS]], bruker informasjonskapsler og lignende sporingsteknologier. Disse retningslinjene for informasjonskapsler forklarer hva informasjonskapsler er, hvordan vi bruker dem, dine valg angående informasjonskapsler, og ytterligere informasjon om informasjonskapsler.

**1. Hva er informasjonskapsler?**

Informasjonskapsler er små tekstfiler som plasseres på enheten din (datamaskin, nettbrett eller mobiltelefon) når du besøker et nettsted. De brukes i stor utstrekning for å få nettsteder til å fungere effektivt, gi analyseinformasjon og aktivere visse funksjoner. Vi bruker også localStorage til visse funksjonelle formål, som beskrevet nedenfor.

**2. Typer informasjonskapsler vi bruker**

**2.1 Strengt nødvendige informasjonskapsler**

Disse informasjonskapslene er essensielle for Tjenestens drift. Uten disse informasjonskapslene kan ikke Tjenesten fungere korrekt. Disse informasjonskapslene krever ikke ditt samtykke.

| Informasjonskapsel                   | Formål                                     | Varighet |
| ------------------------------------ | ------------------------------------------ | -------- |
| `__Secure-better-auth.session_token` | Opprettholder din autentiserte sesjon       | 1 år     |
| Samtykke-informasjonskapsel          | Lagrer dine samtykkepreferanser             | 1 år     |

**2.2 Analyse-informasjonskapsler**

Disse informasjonskapslene gjør det mulig for oss å forstå hvordan besøkende samhandler med Tjenesten ved å samle inn og rapportere informasjon. Vi bruker disse innsiktene til å forbedre Tjenesten. Disse informasjonskapslene lastes via Google Tag Manager (GTM).

| Informasjonskapsel | Formål                                                              | Varighet |
| ------------------ | ------------------------------------------------------------------- | -------- |
| `_ga`              | Google Analytics — skiller mellom unike brukere                     | 2 år     |
| `_gid`             | Google Analytics — skiller mellom unike brukere innenfor en sesjon  | 24 timer |
| `_gat`             | Google Analytics — begrenser forespørselshastigheten                | 1 minutt |
| `_gac_*`           | Google Analytics — lagrer kampanjeinformasjon                       | 90 dager |

**2.3 Funksjonelle informasjonskapsler**

Disse informasjonskapslene muliggjør forbedret funksjonalitet og personalisering.

| Informasjonskapsel | Formål                                                                                                      | Varighet |
| ------------------ | ----------------------------------------------------------------------------------------------------------- | -------- |
| Temapreferanse     | Lagrer ditt foretrukne fargetema (lyst/mørkt)                                                               | 1 år     |
| Språkpreferanse    | Lagrer ditt foretrukne språk/locale                                                                         | 1 år     |

**2.4 Lokal lagring**

Vi bruker også nettleserens localStorage (ikke HTTP-informasjonskapsler) til følgende formål:

| Nøkkel             | Formål                                                         |
| ------------------- | -------------------------------------------------------------- |
| Temainnstilling     | Bevarer lys/mørk-modus-preferansen din på tvers av sesjoner   |
| Brukerpreferanser   | Lagrer UI-preferanser som sidepaneltilstand                    |

**3. Tredjeparts informasjonskapsler**

Noen informasjonskapsler plasseres av tredjepartstjenester som vises på våre sider. Vi kontrollerer ikke disse informasjonskapslene. Tredjeparter som kan sette informasjonskapsler inkluderer:

- **Google Analytics / Google Tag Manager** — analyse og konverteringssporing
- **Leverandør av samtykkeløsning** — håndtering av samtykke for informasjonskapsler
- **Betalingsbehandlere** — funksjonelle informasjonskapsler påkrevd under betalingsbehandling

Hver tredjepartstjeneste har sine egne retningslinjer for personvern og informasjonskapsler. Vi oppfordrer deg til å gjennomgå dem.

**4. Håndtering av samtykke for informasjonskapsler**

Når du besøker Tjenesten for første gang, vil et samtykkebanner bli vist. Du kan velge å godta eller avvise ikke-essensielle kategorier av informasjonskapsler.

Du kan endre preferansene dine for informasjonskapsler når som helst ved å klikke på preferansewidgeten for informasjonskapsler som forblir tilgjengelig på Tjenesten.

Du kan også kontrollere informasjonskapsler gjennom nettleserinnstillingene:

- **Chrome:** Innstillinger > Personvern og sikkerhet > Informasjonskapsler
- **Firefox:** Innstillinger > Personvern og sikkerhet > Informasjonskapsler
- **Safari:** Valg > Personvern > Informasjonskapsler
- **Edge:** Innstillinger > Personvern, søk og tjenester > Informasjonskapsler

Vær oppmerksom på at deaktivering av strengt nødvendige informasjonskapsler kan hindre Tjenesten i å fungere korrekt.

**5. Hvordan trekke tilbake samtykke**

Du kan trekke tilbake samtykket ditt for ikke-essensielle informasjonskapsler når som helst ved å:

- Klikke på preferansewidgeten for informasjonskapsler på Tjenesten
- Slette nettleserens informasjonskapsler og besøke Tjenesten på nytt (et nytt samtykkebanner vil vises)
- Justere preferansene dine for informasjonskapsler i nettleserinnstillingene

**6. Endringer i disse retningslinjene for informasjonskapsler**

Vi kan oppdatere disse retningslinjene for informasjonskapsler fra tid til annen. Vi vil varsle deg om vesentlige endringer ved å publisere de nye retningslinjene på denne siden og oppdatere "Revisjonsdato" øverst.
