## Il Suo CMS è pronto!

Questa è una pagina di esempio creata dallo script di inizializzazione. Può modificarla o eliminarla dal [pannello di amministrazione](/dashboard/cms/pages).

### Per iniziare

- Crei pagine e articoli del blog dalla dashboard
- Carichi file multimediali nella libreria dei media
- Configuri le impostazioni del sito nella sezione Impostazioni
- Gestisca utenti e ruoli dalla sezione Utenti
- Imposti categorie ed etichette per organizzare i contenuti

Consulti il [blog](/blog) per gli ultimi articoli o esplori il [portfolio](/portfolio) per vedere le presentazioni dei progetti.
